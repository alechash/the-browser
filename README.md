# The Browser

A simple, native web browser built with Swift and SwiftUI.

## Features

- Native WebKit browser engine
- Clean SwiftUI interface
- Split view tab support
- Custom home page
- Search engine options
- History tracking

## Requirements

- macOS 11.0+ or iOS 14.0+
- Xcode 13.0+ (for development)

## Development

1. Clone the repo
```bash
git clone https://github.com/alechash/the-browser.git
```

2. Open in Xcode and run
